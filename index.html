Welcome to EA PDF

Measure & mark up drawings for energy assessment

A PDF reader built for scaling measurements, U-value build-ups and psi-junction take-offs. Open a drawing, set the scale (it often auto-detects), measure, and add clouds, sticky notes and comments.

1 Open a PDF or image, or drop one onto the window
2 Check the scale — it auto-detects "1:50" text, or use Calibrate on a known length
3 Use the Measure tools; tag results (Wall, Window…) for the build-up summary
4 Export a flattened PDF, CSV or build-up summary from the File menu
💡 Explain mode — click any tool, field or box to see what it does.

Material λ — thermal conductivity lookup

Indicative λ values (W/m·K) — always confirm against the current product datasheet / BBA certificate; values change with product revisions. New product? Add it yourself (it’s saved on this device) or use the live web search above.

Read written build-up

Paste the architect's wall/floor/roof build-up exactly as written (one layer per line, or separated by commas / slashes). Include thicknesses, e.g. "102.5mm facing brick, 100mm clear cavity, 100mm Celotex CW4000, 100mm blockwork, 12.5mm plasterboard".

Choose a construction for this build-up
